# KernelDex

A community-curated index of GPU kernel implementations.

Right now there's no single place to find GPU kernels. They're scattered across framework repos (vLLM, SGLang, TRT-LLM), vendor libraries (CUTLASS, CK, AITER), competition submissions, personal repos, and blog posts. If you want to find "every MLA decode kernel" or "every FP8 GEMM on MI300X," you're doing archaeology.

KernelDex fixes that. Search by algorithm, hardware target, language, and source.

## Status

Early prototype. ~150 kernels indexed so far (AMD-heavy). Expanding coverage.

## Contributing

We need help cataloging kernels across all hardware targets. If you know where good kernels live, open an issue or PR.

Born from the [GPU MODE](https://discord.gg/gpumode) community.
